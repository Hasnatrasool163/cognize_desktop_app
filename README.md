Cognize Desktop App

A Flutter-based Windows desktop application for managing and interacting with Cognize platform features.

Install the App (Windows)

You must trust the certificate first or Windows will block installation.

Install the certificate

Download and double-click:
test_certificate.cer

Choose:

Local Machine

Place in:
✔ Trusted Root Certification Authorities
✔ Trusted People (optional, recommended)

Install the App

Download and double-click:
cognize_desktop_app.msix

Then click:
✔ Install
✔ Launch
